## 📦 Installation

### Clone the repository
```bash
git clone https://github.com/your-username/real-time-chat-app.git
cd real-time-chat-app
```

### Install dependencies

**Server:**
```bash
cd server
npm install
```

**Client:**
```bash
cd client
npm install
```

### Run the app

**Server:**
```bash
npm start
```

**Client:**
```bash
npm run dev
```
